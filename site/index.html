Bright Screen Flash Alert

The tech streamers I watch in Software and Game Dev tend to use dark mode. It's not infrequent that they hit pages that are considerally brighter when looking up documentation. I thought it would be fun to trigger a little sound effect for the stream when that happens.

The two scripts below set up a quick way to do that on a mac. Pull requests for how to get it working on windows are welcome.

The Scripts

Script 1: watch-screen.bash


#!/bin/bash

ffmpeg -f avfoundation -i "1:0" -vf "fps=2,scale=400:-2" -y images/%04d.png

Script 2: yell-when-it-gets-bright.bash


#!/bin/bash

ALERT_THROTTLE=0

fswatch -0 images/ | while read -d "" event
do 
    if [ -e "$event" ]
    then
        LIGHT_LEVEL_FLOAT=`convert "$event" -colorspace gray -format "%[fx:100*mean]" info:`
        LIGHT_LEVEL_INT=`printf %.0f "$LIGHT_LEVEL_FLOAT"`
        echo $LIGHT_LEVEL_INT
        if [ $LIGHT_LEVEL_INT -gt 40 ]
        then
            if [ $ALERT_THROTTLE -eq 0 ]
            then
                ALERT_THROTTLE=1
                afplay ahhhh.mp3
                echo "YELL"
            fi
        else
            ALERT_THROTTLE=0
        fi
        rm "$event"
    fi
done

Prerequisites

You'll need `ffmpeg`, `imagemagick`, and `fswatch`. They can be installed with these Homebrew commands if you don't already have them

brew install ffmpeg
brew install imagemagick
brew install fswatch

Setup

The ffmpeg command in the script watches a specifics screen. To tell it which one to watch, run this command: 


ffmpeg -f avfoundation -list_devices true -i ""

Usage

  1. Copy the code for the scripts above into two seperate files in the same directory. One for `watch-screen.bash` and one for `yell-when-it-gets-bright.bash`
  2. Make the scripts executable from a terminal command line with:

    chmod u+x watch-screen.bash

    and

    chmod u+x yell-when-it-gets-bright.bash
  3. Download the mp3 file from here and put it in the same directory as the scripts.
  4. Make an `images` directory inside the directory you have your scripts
  5. Run the scripts from the command line with

    ./yell-when-it-gets-bright.bash
    ./watch-screen.bash

If all goes well, you'll hear the sound trigger each time your screen crosses a certain brightness threshold.

Press `CTRL c` on each of the terminal windows when you're ready to stop the process.
