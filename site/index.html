Bright Screen Flash Alert

The tech streamers I watch in Software and Game Dev tend to use dark mode. It's not infrequent that they're looking up documentation and hit a page that's considerably brighter. I thought it would be fun to trigger a little sound alert when that happens.

These two scripts set up a quick way to do that on a mac. Pull requests welcome for making it work on windows.

The Scripts

Script 1: watch-screen.bash


#!/bin/bash

ffmpeg -f avfoundation -i "1:0" -vf "fps=2,scale=400:-2" -y images/%04d.png

Script 2: yell-when-it-gets-bright.bash


#!/bin/bash

ALERT_THROTTLE=0

fswatch -0 images/ | while read -d "" event
do 
    if [ -e "$event" ]
    then
        LIGHT_LEVEL_FLOAT=`convert "$event" -colorspace gray -format "%[fx:100*mean]" info:`
        LIGHT_LEVEL_INT=`printf %.0f "$LIGHT_LEVEL_FLOAT"`
        echo $LIGHT_LEVEL_INT
        if [ $LIGHT_LEVEL_INT -gt 40 ]
        then
            if [ $ALERT_THROTTLE -eq 0 ]
            then
                ALERT_THROTTLE=1
                afplay ahhhh.mp3
                echo "YELL"
            fi
        else
            ALERT_THROTTLE=0
        fi
        rm "$event"
    fi
done

Usage

  1. Install `ffmpeg`, `imagemagick`, and `fswatch` which with Homebrew if you don't already have them
  2. Copy the code for the scripts above into two seperate files in the same directory. One for `watch-screen.bash` and one for `yell-when-it-gets-bright.bash`
  3. Make the scripts executable from a command line terminal with:

    chmod u+x watch-screen.bash

    and

    chmod u+x yell-when-it-gets-bright.bash`
